# 🐍 Portfólio de Automação e Análise de Dados em Python

## 📝 Descrição do Projeto
Este repositório reúne uma suíte de scripts desenvolvidos para automatizar fluxos de decisão lógica em ambientes corporativos e de monitoramento ambiental. O foco do projeto é a implementação de algoritmos robustos que lidam com validação de dados, cálculos estatísticos simples e automação de regras de negócio[cite: 4, 5].

O projeto é composto por dois módulos principais:
1.  **Módulo de Processamento de Vendas:** Um sistema de PDV (Ponto de Venda) lógico que gerencia inventários temporários e aplica políticas de desconto dinâmicas[cite: 4].
2.  **Módulo de Monitoramento Climático:** Uma ferramenta de análise de dados meteorológicos para detecção de anomalias térmicas semanais[cite: 5].

## ✨ Funcionalidades Principais

### 🛒 Gestão de Vendas e Descontos
*   **Entrada Dinâmica:** Permite o cadastro de múltiplos tipos de produtos em uma única sessão[cite: 4].
*   **Validação de Integridade:** Verifica se preços e quantidades são valores positivos antes do processamento[cite: 4].
*   **Motor de Descontos Progressivos:**
    *   **10% Off:** Para compras acima de R$ 500,00[cite: 4].
    *   **5% Off:** Para compras entre R$ 200,00 e R$ 500,00[cite: 4].
*   **Tratamento de Exceções:** Sistema protegido contra entradas não numéricas via blocos `try-except`[cite: 4].

### 🌡️ Análise Climática Semanal
*   **Processamento de Séries Temporais:** Coleta e processa temperaturas ao longo de um ciclo de 7 dias[cite: 5].
*   **Cálculo de Média Operacional:** Determina a temperatura média semanal para relatórios de rotina[cite: 5].
*   **Alerta de Condições Extremas:** Identifica automaticamente "dias quentes" (> 35°C) e dispara avisos de segurança caso a frequência de calor extremo seja detectada[cite: 5].

## 🚀 Tecnologias Utilizadas
*   **Linguagem:** Python 3.x[cite: 4, 5]
*   **Ferramentas:** Google Colab, Jupyter Notebook[cite: 4, 5]
*   **Conceitos Aplicados:** Estruturas de repetição (`for`), Condicionais compostas, e Manipulação de tipos primitivos (`float`, `int`, `str`)[cite: 4, 5].

## 📊 Resultados e Aprendizados
O desenvolvimento destes scripts permitiu consolidar boas práticas de programação defensiva e lógica estruturada:
*   **Programação Defensiva:** Implementação de travas de erro para evitar falhas de execução por input do usuário[cite: 4].
*   **Otimização de Fluxo:** Uso de acumuladores e flags lógicas para monitoramento de estados complexos (como o `alerta_extremo`)[cite: 5].

## 🔧 Como Executar
1. **Ambiente:** Recomenda-se o uso do Google Colab para execução imediata das células[cite: 4, 5].
2. **Módulo de Vendas:** Chame a função `processar_vendas()` e siga as instruções do prompt para inserir produtos, preços e quantidades[cite: 4].
3. **Módulo Climático:** Chame a função `analisar_clima()` e insira as 7 temperaturas diárias conforme solicitado[cite: 5].
